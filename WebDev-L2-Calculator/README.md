## 📌 Overview

A fully functional interactive calculator built with **HTML, CSS, and JavaScript**. This is **Task 1** of my Web Development Level 2 internship at **Oasis Infobyte**.

The calculator supports basic arithmetic operations, operator chaining, and includes error handling for division by zero.

## 🎮 How to Use

### Using Mouse/Touch:
1. Click number buttons to enter digits
2. Click an operator (+, −, ×, ÷)
3. Continue with more numbers and operators
4. Click `=` to see the result
5. Click `C` to clear everything
6. Click `←` to delete the last digit

### Using Keyboard:
| Key | Action |
|-----|--------|
| `0-9` | Enter numbers |
| `.` | Add decimal point |
| `+`, `-`, `*`, `/` | Operators |
| `Enter` or `=` | Calculate |
| `Backspace` | Delete last digit |
| `Escape` or `Delete` | Clear all |
