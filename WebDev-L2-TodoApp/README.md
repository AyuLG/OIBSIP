## 📌 Overview

An interactive to-do list application that allows users to manage daily tasks with add, complete, edit, and delete functionality. This is **Task 3** of my Web Development Level 2 internship at **Oasis Infobyte**.

Tasks are organized into **Pending** and **Completed** lists with persistent storage using localStorage.

## 🎮 How to Use

### Add a Task
1. Type your task in the input field
2. Click **"Add"** button or press **Enter** key

### Complete a Task
- Click the circle checkbox next to a task
- The task moves to the **Completed** list

### Edit a Task
1. Click the ✏️ (pen) icon on any task
2. Edit the text in the input field
3. Click **"Save"** or press **Enter** to confirm
4. Click **"Cancel"** or press **Esc** to discard changes

### Delete a Task
- Click the 🗑️ (trash) icon to permanently remove a task
